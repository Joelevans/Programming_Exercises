Correspondence btwn C# and .NET Types
NB: >> to mean corresponds
int in C# >> System.Int32 in CTS >> Integer type in VB.NET
long in C# >> System.Int64 in CTS >> Long type in VB.NET

Due to the common types system (CTS) in .NET Framework,
there is compatibility between prog langs (C#, C++, VB.NET
and F#)

int, Int32 and System.Int32 types are diff aliases of
signed 32-bit integer.
